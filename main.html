This is about me

주도적으로 문제를 해결하고, 넓은 개발 시야를 가진 퍼블리셔 곽호선입니다. 정보통신공학을 전공하고, 백엔드(2년)와 퍼블리셔(4년) 실무 경험을 쌓아왔습니다. 퍼블리셔로서 플랫폼 기업(2년)과 웹 에이전시(2년)에서의 경험 을 통해, 자사 콘텐츠를 기반으로 한 업무와 고객사의 요청을 처리하는 업무를 모두 경험 했습니다. 이를 통해 다양한 직군과의 협업 능력뿐만 아니라, 고객과의 원활한 소통 역량도 갖출 수 있었습니다.
특히 에이전시에서의 경험을 통해 다양한 고객과 수많은 코드들을 접하며 빠르게 성장할 수 있었습니다. 또한, 새로운 기술 습득을 위해 패스트캠퍼스 부트캠프(4개월)에서 React를 집중적으로 학습하며 프론트엔드 역량을 더욱 강화했습니다. 앞으로도 마크업을 하듯 구조를 탄탄히 다지며, 끊임없이 배우고 도전하는 퍼블리셔로 성장해 나가겠습니다.

jQuery/javaScript
자바스크립트는 동적이고 명령어, 객체 지향, 함수 프로그래밍 스타일을 지원하는 언어입니다. 클라이언트 측 웹(브라우저)에서 실행 되고, 웹 페이지가 이벤트 발생 시 어떻게 작동하는 지 디자인/프로그래밍할 수 있습니다. 제이쿼리는 오픈 소스 기반의 자바스크립트 라이브러리입니다.
제이쿼리는 자바스크립트 라이브러리중에서 제일 많이 사용되고 있기 때문에 사용했습니다. 자바스크립트와 제이쿼리를 이용하여 마우스의 움직임을 이용한 마우스 효과, 스크롤에 따른 패럴랙스 효과, 슬라이더 효과를 모아 예제 페이지를 만들어 필요한 기능이 있을 때 바로 참고할 수 있게 만들었습니다.
animation
애니메이션은 사이트에 동적인 움직임을 주어 생동감을 줍니다. CSS를 활용하여 간단한 애니메이션을 만들 수 있고, 자바스크립트를 추가하면 CSS 애니메이션을 조작할수 있고, 더 효과적인 애니메이션을 만들 수 있습니다.
transition과 @keyframe을 적절히 활용하여 애니메이션을 만듭니다. 애니메이션은 codepen을 활용하여 제작하였습니다. 아래에는 codepen을 직접 삽입하여 코드를 확인할 수 있게 하였습니다.
